Glossary

A description of some cryptomining terminology

Wallet address

When you set up a monero wallet to hold your monero you'll have an associated public wallet address.This is used to transfer monero across accounts.We require you to enter this so we can transfer monero payments to your account.

Throttle

This determines how much of the user's CPU power is used to mine monero.This is not an exact amount but a rough estimate.At higher percentages we use more of your visitors cpu to mine coins. Monitoring cpu usage will show spikes as the throttle determines how frequently the cpu mines. As a result on different devices the same throttle could spike to different percentages of cpu. We do have an option to disable on mobile devices to safegaurd against this aspect.

Mining pool

This is a shared mining area where users contribute(pool) their CPU power to try solve a mathematical calculation(block).In simpler terms,when the pool solves one of these calculations they receive monero,which is split among the pool based on the amount each user contributed.

Monero

Name of the cryptocurrency being mined by this miner.Different cryptocurrencies use different hashing algorithms and have different wallets.
