How to use the monero miner

  1. 1. Setup a Monero wallet with mymonero.com
  2. 2. Copy your public wallet address
  3. 3. Go to the Cryptonate Miner Code Generator
  4. 4. Paste your address into the Wallet ID section, choose a Throttle Value and enter your website address
  5. 5. Place the generated code at the top of the webpage you wish the miner to run on
  6. 6. [OPTIONAL] Place the Opt-In pop-up code on your webpage
  7. 7. Mine away!
